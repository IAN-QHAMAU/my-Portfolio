# Ian's Learning Lab - Interactive Portfolio

A living lab notebook documenting my learning journey as a first-year, first-semester Data Science student at the University of Nairobi (UON). This isn't a résumé—it's a thinking-first portfolio that shows how I learn, reflect, and grow.

## 🎯 Purpose

This portfolio emphasizes:
- **Thinking First**: How I approach problems and learn new concepts
- **Learning Timeline**: My journey through the first semester
- **Case-Note Projects**: Reflective documentation of what I'm building
- **AI Insights**: What I'm learning about modern AI with honest reflections
- **Honest Tool Stack**: What I know, what I'm learning, what I'm still figuring out
- **Thinking Artifacts**: My problem-solving approaches and mental models
- **Curiosity**: Questions I'm exploring and things I want to learn
- **Intentional Contact**: Meaningful connections over networking

## 🚀 Features

- **Mobile-First Design**: Clean, minimal interface optimized for all devices
- **Living Notebook Feel**: Journal-style layout that evolves as I learn
- **Interactive Elements**: Smooth scrolling, animations, and engaging transitions
- **Reflection-Focused**: Every section includes honest reflections on learning
- **Accessibility**: Keyboard navigation and semantic HTML

## 🛠️ Technologies Used

- **HTML5**: Semantic structure
- **CSS3**: Custom properties, Grid, Flexbox, animations
- **Vanilla JavaScript**: Smooth interactions, scroll animations, intersection observers

## 🎨 Design Philosophy

- **Minimal**: Clean, distraction-free design
- **Readable**: Typography optimized for long-form content
- **Lab Notebook**: Feels like a personal journal, not a corporate website
- **Thoughtful**: Every element serves the purpose of showing learning process

## 🌱 About This Project

Built as part of my learning journey in web development and data science. This portfolio itself is a learning artifact—I built it to understand HTML, CSS, and JavaScript while showcasing my broader learning in data science and AI.
